DispatcherCreate

Creates a new callback dispatcher

Make sure to delete dispatchers when done with them

DispatcherCreate()

Returns: dispatcher id

Example:

newDispatcher = DispatcherCreate();


DispatcherDestroy

Destroys the indicated dispatcher

DispatcherDestroy(dispatcher)

dispatcher

previously created dispatcher

Returns: null

Example:

dispatcher = DispatcherCreate();

DispatcherDestroy(dispatcher);

dispatcher = DispatcherNull();


DispatcherAdd

Adds a callback to the designated dispatcher ** Script callbacks can be supplied up to 13 arguments **

DispatcherAdd(dispatcher,target,script,[arg0,arg1,...])

dispatcher

previously created dispatcher

target

instance to execute callback

script

script to call

arg0....

argument to pass to script

Returns: callback id

Example:

dispatcher = DispatcherCreate();

callback = DispatcherAdd(dispatcher, id, ShowMessage, “Hello!”);


DispatcherInsert

DispatcherInsert(dispatcher,target,script,[arg0,arg1,...])

dispatcher

previously created dispatcher

target

instance to execute callback

script

script to call

arg0....

argument to pass to script

Returns: callback id

Example:

DispatcherInvoke

Executes all callbacks belonging to the indicated dispatcher

DispatcherInvoke(dispatcher)

dispatcher

previously created dispatcher

Returns: value returned from first callback

Example:

dispatcher = DispatcherCreate();

DispatcherAdd(dispatcher, id, ShowMessage, “Hello!”);

DispatcherInvoke(dispatcher);


DispatcherInvokeExt

Executes all callbacks belonging to the indicated dispatcher and overrides script arguments ** Script callbacks can be supplied up to 13 arguments

DispatcherInvokeExt(dispatcher,[arg0,arg1,...])

dispatcher

previously created dispatcher

arg0...

arguments to override existing script arguments

Returns: value returned from first callback

Example:

dispatcher = DispatcherCreate();

DispatcherAdd(dispatcher, id, ShowMessage, “Hello!”);

DispatcherInvokeExt(dispatcher, “Bye!”);

DynamicEventsRegister

Declares that the indicated target object/instance will use dynamic events – Typically called in Create Event

Make sure to call DynamicEventsUnregister() when object is destroyed or when room ends

DyanmicEventsRegister(target)

target

instance id

Returns: na

Example:

/// Object Create Event

DynamicEventsRegister(id);


DynamicEventsUnregister

Invalidates and destroys dynamic events for indicated target object/instance

Make sure to call this when an object is destroyed or when room ends

DyanmicEventsUnregister(target)

target

instance id

Returns: na

Example:

//// Create Event

DynamicEventsRegister(id);

/// Destroy Event

DynamicEventsUnregister(id);

/// Room End Event

DynamicEventsUnregister(id);


DynamicEventAdd

DynamicEventAdd(target,type,numb,script,[arg0,arg1,...])

DynamicEventInvoke

Executes callbacks associated with indicated event

To call events from other instances, please use with(other) DynamicEventInvoke()

DynamicEventInvoke(type,numb)

type

event type (e.g. ev_mouse)

numb

event number (e.g. ev_left_button)

Returns: value returned from first callback

Example:

/// Create Event

DynamicEventsRegister(id);

DynamicEventAdd(id, ev_keypress, vk_enter, ShowMessage, “Enter Pressed!”);

// Keypress Enter Event

DynamicEventCall(ev_keypress, vk_enter);
